App for CommonsBooking<\/a> API<\/h2>\n\n\n\n

This app offers the possibility to access functions of a website that is operated with the CommonsBooking plugin.<\/p>\n\n\n\n

For read-only access, it is sufficient if the CommonsAPI is activated.<\/p>\n\n\n\n

With the add-on plugin „API for CB App“ it is possible to log in as a user and make bookings.<\/p>\n<\/div>\n\n\n\n
