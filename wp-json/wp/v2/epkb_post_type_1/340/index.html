The app can also be used to a limited extent without an internet connection. For this purpose, part of the data is saved locally for each online connection to a service provider.
(This behavior can be deactivated for each service provider in the settings <\/a>(e.g. to save local storage space))<\/em><\/p>\n\n\n\n


Bookings cannot be made in offline mode. Accordingly, the availability calendar is displayed in read-only mode.<\/p>\n\n\n\n


The booking list is also available in read-only mode, i.e. the information can be viewed but a booking cannot be canceled (the time at which the data was last updated is displayed in the top line).<\/p>\n\n\n\n


You can tell whether the app is in offline mode by the fact that the icon next to the main menu button has changed. The date and time when the data was last loaded online is also displayed there.<\/p>\n\n\n\n
