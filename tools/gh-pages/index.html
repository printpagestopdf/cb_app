App für CommonsBooking API

Diese App bietet die Möglichkeit auf Funktionen einer Website, die mit dem CommonsBooking Plugin betrieben wird zuzugreifen.

Für reinen Lesezugriff ist es ausreichend, wenn die CommonsAPI aktiviert ist.

Mit dem AddOn Plugin "API for CB App" ist es möglich sich als Benutzer anzumelden und Buchungen vorzunehmen.
